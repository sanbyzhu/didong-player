# 洞听播放器 - 产品规格

## MVP 范围
1. 安卓原生 App，名称“洞听播放器”。
2. 使用 AndroidX Media3/ExoPlayer 播放本地音频、视频。
3. 通过系统文件夹选择器扫描目录，识别常见音视频扩展名。
4. 文件夹播放：按文件名自然排序；含数字时按数字优先排序。
5. 播放列表：可新建列表，把扫描到的文件加入列表并按列表播放。
6. 播放记忆：
   - 每个媒体文件记住播放位置。
   - 倍速默认使用同一文件夹、同一媒体类型上次使用值。
   - 音频与视频倍速相互独立。
7. 倍速控制：0.25x 至 8.0x。
8. 循环模式：关闭、单曲循环、列表循环。
9. AB 循环：可设置 A 点、B 点、清除；按媒体文件记忆，下次打开该媒体默认生效，直到用户清除或换媒体。
10. 音量/音质：
    - App 内音量滑杆。
    - LoudnessEnhancer 增益，用于不修改源文件的音量增强。
    - 音频增强开关，优先启用系统可用的响度增强，并采用高质量音频属性。
11. 文本朗读：
    - 导入 `.txt` 文本。
    - 从系统 TTS 引擎列出可用语音。
    - 朗读时可选择一个本地音频/视频作为背景音乐并调节背景音量。
12. 二期预留：Windows 端和荣耀投影/无线投屏。

## 非目标
- 首版不做账号、云同步、在线播放源管理。
- 首版不承诺所有罕见封装/编码格式；以 Android 平台解码能力和 Media3 支持为准。
- 首版不内置商业 TTS 语音包，避免授权问题。

## 技术选择
- Android Java + Android Gradle Plugin。
- AndroidX Media3 ExoPlayer + UI。
- DocumentFile 处理 SAF 目录扫描。
- SharedPreferences + JSON 作为轻量持久化。
- Android TextToSpeech 朗读文本。
